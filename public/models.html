Configuration Guide

                    

                        

                            
Environment Variables Required:

                            
    
                                
  • OPENAI_API_KEY - OpenAI API key for GPT models
    • 
                                
  • CLAUDE_API_KEY - Anthropic API key for Claude models
    • 
                                
  • GEMINI_API_KEY - Google API key for Gemini models
    • 
                                
  • GROK_API_KEY - xAI API key for Grok models
    • 
                            

                        

                        

                            
Model Switching:

                            
To switch models, update the model ID in analyzeCommit.js and restart the service. 
                            Use this page to verify the model is working before committing changes.

                        

                        

                            
Cost Management:

                            
Monitor token usage and costs here. Set up alerts when costs exceed thresholds. 
                            Gemini offers the lowest cost, while o3-mini has premium pricing.
